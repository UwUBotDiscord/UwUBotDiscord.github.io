Commands

-General Commands-

/commands - show commands

/help - show help/info

/info - show help/info

/website - shows bot website

/serverstats - shows server infomation

/UwU - posts a UwU to chat

/OwO - posts a OwO to chat

/reddit - posts the reddit link to chat

/cat - posts a cat image to chat

/fox - posts a fox image to chat

/proto - posts a protogen image to chat

/boop [user] - boops a user

/hug [user] - hugs a user

/say [message] - sends a message to chat

/comedymoment - gives a award for comedy

/delet-this - delet this gif

/cookie - posts oOoOOo a cookie :3 to chat

/botinvite - posts bot invite link to chat

/dice - picks a number including or between 1 and 6

/gay - picks a percentage on how gay you are out of 100

/furry - picks a percentage on how much of a furry you are out of 100

/ship [user] - ships you with a user and gives a percentage on compatiblity out of 100

/global [message] - posts a message in all servers with #global channel

/topgg - gives link to top.gg page

-Mod Commands-

/kick [user] - kicks a user from server

/ban [user] - bans a user from server

/warn [user] [message] - warns a user

/clear - clears 25 messages



Create a #uwulog channel to log messages

Create a #welcome for welcome cards
