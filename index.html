-General Commands-

uwucommands - show commands

uwuhelp - show help/info

uwuinfo - show help/info

uwudiscord - official uwubot discord link

uwuwebsite - shows bot website

uwubotstats - shows bot infomation

uwuUwU - posts a UwU to chat

uwuOwO - posts a OwO to chat

uwureddit - posts the reddit link to chat

uwucat - posts a cat image to chat

uwufox - posts a fox image to chat

uwuproto - posts a protogen image to chat

uwuboop [user] - boops a user

uwuhug [user] - hugs a user

uwusay [message] - sends a message to chat

uwucomedymoment - gives a award for comedy

uwudelet-this - delet this gif

uwucookie - posts oOoOOo a cookie :3 to chat

uwubotinvite - posts bot invite link to chat

uwudice - picks a number including or between 1 and 6

uwugay - picks a percentage on how gay you are out of 100

uwufurry - picks a percentage on how much of a furry you are out of 100

uwuship [user] - ships you with a user and gives a percentage on compatiblity out of 100

uwuglobal [message] - posts a message in all servers with #global channel

uwuimgglobal [attachment] - posts a image in all servers with #global channel

uwutopgg - gives link to top.gg page

uwurank [optional user] - gives rank of user

uwuleaderboard - gives the leaderboard for current server

-Mod Commands-

uwukick [user] - kicks a user from server

uwuban [user] - bans a user from server

uwuwarn [user] [message] - warns a user

uwuclear [amount] - clears messages

uwuloghelp - help for #uwulog

 

Required permissions: SEND_MESSAGES, READ_MESSAGE_HISTORY, ATTACH_FILES, EMBED_LINKS, KICK_MEMBERS, BAN_MEMBERS, MANAGE_MESSAGES, VIEW_CHANNEL, USE_EXTERNAL_EMOJIS
